способо 3 - билинейная интерполяция 
если определить положение внутренней точки числами a = x - [x] и b = y - [y]. то для расчета подойдут следующие формулы:

Cl = C1 + b(C3 - C1)
Cr = C2 + b(C4 - C2)
Color = Cl + a (Cr - Cl) (здесь имеется ввиду для каждой составляющей цвета)

здесь по закону линейной интерполяции найдены цвета проекций точки P(x,y) на левую сторону внутренней области CL, на правую сотороу CR и по полученным значениям P

способ 4 - бикубическая интерполяция

для определения значений участвуют 16 ближайших точек. по аналогии с билинейной интерполяцией, использовао вычисление значений на пересечении горизонтали (y) с линиями сетки. найденные значения позволяют найти коэффициенты кубического полинома,  значения которого в указанных четырех точках совпадают с интенсивностью соответствующей состовляющей цвета в этих точках

трехмерное вращение вокруг произвольной оси
Ось проходить через начало координат

вращение происходит вокруг оси ON, где точка O является началом координат,  а N - произвольная точка с координатами Nx Ny Nz
поворот выполняется на угол omega

alpha = <(ON, Ox)
beta = <(ON, Oy)
gamma = <(ON, Oz)

обозначим dN - длина вектора ON 
dn = sqrt(Nx^2 + Ny^2 + Nz^2)
направляющие косинусы вектора ON :
n1 = Nx/dN  n2 = Ny/dN  n3 = Nz/dN

полученный вектор является единичным и сонаправлен с вектором ON

матрица поворота вокруг оси ON на угол omega имеет вид
T:
n1^2 + (1-n1^2)cos(omega)            n1n2(1 - cos(omega))+n1sin(omega)   n1n3(1 - cos(omega)) - n2sin(omega)  0
n1n2(1 - cos(omega)) - n3sin(omega)  n2^2 + (1-n2^2)cos(omega)           n2n3(1 - cos(omega)) + n1sin(omega)  0
n1n3(1 - cos(omega)) + n2sin(omega)  n2n3(1 - cos(omega)) - n1sin(omega) n3^2 + (1 - n3^2)cos(omega)          0
                0                                    0                            0                           1
